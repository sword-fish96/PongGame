Néhány fontos adat:

A számitógépen, melyen a program készült a következő szoftverek futottak:

	-os: 			WIN10 x64
	-.NET version:		4.6.1
	-forráskódszerkesztő:	MICROSOFT VISUAL STUDIO 2017

1.Lépés
	Töltsd le a fájlt. 

2.Lépés
	Csomagold ki a rar-t.

3.Lépés
	Indítás:  
		-közvetlen indítás:  \PongGame\bin\Debug\PongGame.exe
		-közvettet inditás:  VISUAL STUDIO 2017-vel a teljes project megnyitása (PongGame.sln), majd azon belül futtatás

	        -Szerkesztés:  VISUAL STUDIO 2017 használatával
		
		
Megjegyzés: Az alkalmazás Windows Form App-ként lett megírva. Az 1972-es Atari által készített "Pong" játék inspirált.
